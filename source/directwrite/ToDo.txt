IDWriteFontCollectionLoader (FontCollectionLoader)
	Client implements this interface to provide a custom font collection loader to
	IDWriteFactory::CreateCustomFontCollection
IDWriteFontFileLoader (FontFileLoader)
	Client implements this interface to provide a custom font file loader to
	IDWriteFactory::CreateCustomFontFileReference
	Returned by IDWriteFontFile::GetLoader
IDWriteFontFileStream (FontFileStream)
	Returned by IDWriteFontFileLoader.
	Could be implemented by the client or the runtime.
IDWriteLocalFontFileLoader
	Stock implementation of IDWriteFontFileLoader for local font files.
	Font file references created using CreateFontFileReference use this font file loader.
